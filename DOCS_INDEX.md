## 📋 文档使用建议

### 首次使用

1. **先读** [README.md](./README.md) - 了解项目整体
2. **再读** [USAGE_GUIDE.md](./USAGE_GUIDE.md) - 学习具体使用方法
3. **需要时读** [DATA_SYNC_GUIDE.md](./DATA_SYNC_GUIDE.md) - 数据同步和备份

### 参与开发

2. 了解项目结构和技术栈
3. Fork 项目并提交 Pull Request
